***Answer 2:
man 1 write выдаёт описание команды write(1), которая является командой системы Linux и используется для отправки сообщения другому пользователю. Команда man 2 write показывает описание Си-функции write(), которая используется для записи данных в файл.
***Answer 3:
cat выполняет конкатенацию содержимого файлов и выводит на стандартный поток
grep выполняет поиск в содержимом файлов (или стандартного потока, если файлы не указаны) данных, которые совпадают с введёнными
head выводит на стандартный поток первые 10 строк из каждого файла
ps выводит информацию о текущих процессах
sort выводит на стандартный поток отсортированную последовательность строк текстовых файлов
wc печатает последовательности байтов и машинных слов для каждого файла
yes печатает строку неограниченное число раз, пока не не завершится вручную
